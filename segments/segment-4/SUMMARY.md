# Segment 4 — Gemma Explainability + What-If Replay
## SUMMARY

**Status:** ✅ Complete — 12/12 tests passed

## What was built

Every flagged or blocked prediction now gets a plain-English explanation generated by Gemma.
The What-If Bias Replay engine lets reviewers flip one sensitive attribute (e.g. gender)
on any blocked candidate and see whether the outcome changes — making bias undeniable.
Both work with or without a real Gemma instance (graceful template fallback in dev).

## Gemma swap ladder (RTX 3050 — 6GB VRAM)

```
PRIMARY   gemma4:e4b  — 6GB VRAM — best quality
FALLBACK1 gemma4:e2b  — 3GB VRAM — slightly smaller
FALLBACK2 gemma3:4b   — 3GB VRAM — always works
FALLBACK3 template    — no GPU   — hardcoded safe response
```

Configured via `.env`:
```
GEMMA_BACKEND=ollama             # local dev
GEMMA_MODEL_PRIMARY=gemma4:e4b
GEMMA_MODEL_FALLBACK1=gemma4:e2b
GEMMA_MODEL_FALLBACK2=gemma3:4b
```

Switch to cloud: `GEMMA_BACKEND=vertex`
